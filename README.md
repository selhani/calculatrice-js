# Calculatrice Simple en ligne

Il s'agit d'une petite calculatrice en **HTML, CSS et JavaScript** pour effectuer des opérations de base comme l'addition, la soustraction, la multiplication et la division.
L'objectif principal de ce projet était de pratiquer la mise en page avec **CSS Grid** et d'appliquer des notions de **JavaScript pour la logique de calcul**.

---

### Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Grid

## Fonctionnalités

- Calculs de base : Addition, soustraction, multiplication et division
- Interface simple et intuitive
- Mise en page réalisée avec **CSS Grid**
- Interaction avec l’utilisateur via **JavaScript**
